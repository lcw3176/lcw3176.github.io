## Index

### 소개
가상 메모리는 메모리 관리 기법 중 하나로, 
각 프로그램에 실제 메모리 주소가 아닌 가상의 메모리 주소를 주는 방식이다.
프로세스가 직접 메모리 주소를 통해 접근하는 것을 방지하여 보안 및 자원 관리에
용이하다.
